How to Use This Dashboard

▼
1. Import Data
  • Export your QC data from your instrument software as a semicolon-delimited CSV file
  • Drag and drop one or more CSV files onto the upload area below, or click to browse
  • Multiple files will be merged automatically
  • Click Process Data to begin
2. Review Results
  • The Results Summary tab shows Mean, SD, CV%, and n for each analyte across all four instruments (AU/DxI-1 to 4) plus a combined column
  • CV% is colour-coded: green (<5%), amber (5-10%), red (>10%)
  • Click any value to see the full calculation breakdown and all data points used
3. Filter & Refine
  • Use the Analyte, Protocol, and Date Range filters to narrow results
  • Open the Instrument Exclusions panel to exclude a specific instrument for a single analyte
  • The Charts tab shows Levey-Jennings control charts with SD lines and a CV% comparison bar chart
4. Export & Save
  • Export XLSX — formatted Excel file matching the original QC Audit template layout
  • Export CSV — plain CSV for further analysis
  • Create Report (PDF) — generates a PDF with summary stats and results table
  • Save Report — saves to the database so you can reload it later via Load Report
Expected CSV format: Semicolon-delimited with columns: Protocol name; Instrument; Parameter; Level; Date; Value; Target; SD; Status; Message; User comment; User; Sample ID
